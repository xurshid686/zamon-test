# Zamonlar Test - English Tenses Test

This is an English tenses test focusing on Simple and Continuous forms with 20 questions.

## Features
- User authentication (name and surname)
- 20 questions about Simple and Continuous tenses
- Real-time results
- Telegram notifications
- Deployable on Vercel

## Setup
1. Upload to GitHub
2. Deploy on Vercel
3. Configure Telegram bot in api/telegram.js
